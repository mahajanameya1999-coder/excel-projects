F AND P Sales Analysis Dashboard
This dashboard provides a comprehensive overview of sales performance, highlighting key metrics such as total orders, total revenue, average delivery time, and customer spending. It includes visual insights like revenue by occasion, top-performing cities, day-wise and month-wise trends, and category-wise distribution. Interactive filters allow users to explore data by occasion and date, enabling quick and informed business decisions.
